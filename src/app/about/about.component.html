Angular Blogging Web App

Descriere

Acest proiect constă într-o aplicație web de blogging construită cu Angular pentru frontend și PHP pentru backend. Aplicația permite utilizatorilor să citească și să publice articole pe diverse teme.

Setup

Frontend (Angular)

  1. Instalare Dependințe:
    2. cd frontend
npm install
  2. Rulează Serverul de Dezvoltare:
    3. ng serve

    Accesează aplicația într-un browser la adresa http://localhost:4200/.

Backend (PHP)

  1. Instalare PHP și Composer:

    2. Asigură-te că ai PHP și Composer instalate.

  2. Instalare Dependințe:
    3. cd backend
composer install
  3. Configurare Bază de Date:

    4. Configură fișierul config.php pentru a se potrivi cu setările bazei de date.

  4. Rulează Serverul PHP:
    5. php -S localhost:8000

    Accesează API-ul la http://localhost:8000/api.

Structura Proiectului

- frontend/: Codul sursă Angular pentru frontend.

- backend/: Codul sursă PHP pentru backend.

- docs/: Documentație suplimentară.

Tehnologii Utilizate

- Angular

- PHP

- MySQL (sau altă bază de date la alegere)
